# REPO-VERWALTUNG / INFOS

Dieses Repository ist der zentrale Info-, Hash- und Abgleich-Knoten für BIG SAVE.

BIG SAVE / `D:\` ist das lokale Hauptsystem. Dieses Repo enthält nur Metadaten, keine Projektinhalte.

## Zweck

- Hash- und Speicherabgleich
- SSD-Inventar
- Repo-Mapping
- GitHub-/Desktop-/BIG-SAVE-Abgleich
- KI-/Copilot-/Agent-Handoff
- Vollabgleich-Protokolle

## Wichtig

Trading-Zentrale ist **nicht** der Mittelpunkt der SSD. Trading-Zentrale ist nur ein Projekt innerhalb von BIG SAVE.

SSD-weite Informationen gehören in dieses Repo, nicht in einzelne Projekt-Repos.
